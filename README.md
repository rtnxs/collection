## Система составления/сохранения информации о коллекции

## Не должно быть ограничений по добавляемым типам коллекционируемых предметов (деньги, марки, крышки от пива, и пр)
## Регистрация пользователей
## Посмотреть коллекцию других пользователей могут посмотреть только зарегистрированные пользователи
## Должна быть группировка/фильтрация
## Должна быть возможность распечатать QR код как для отдельного элемента, так и для группы (оставить на потом)
## Поля
- название 
- тип
- город производитель
- год производства
- изображение
- стоимость покупки
- комментарий
- текущая цена
- QR код со всей информацией (оставить на потом)

## текущая цена - настраиваемое поле/парсер который в реальном времени находит предмет коллекции на соответствующем сайте и забирает цену
